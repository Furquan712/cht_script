AI Assistant Demo

A small demo page showing the injected assistant widget.

Welcome

Open the assistant using the floating button in the lower-left. This demo uses the local fallback reply by default.

Optional: configure a backend

If you have a server you want the widget to call, set window.ChatbotConfig before the script loads. Example:

window.ChatbotConfig = {
  apiUrl: 'https://your-api.example.com/chat',
  apiKey: 'sk-...'
};

Place that configuration in a script tag above the included script.js if you want the widget to call your API.

Try it

Click the floating button, type a message, and press Enter to send. The widget will echo your message in this demo.
